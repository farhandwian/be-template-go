# Automatic Water Level Radar (AWLR)

Jumlah unit: **613 unit**

## Komponen AWLR

### 1. Sensor Radar
- **Fungsi:** Sensor radar digunakan untuk mengukur ketinggian muka air berdasarkan pembacaan pantulan gelombang radar.

### 2. Panel Box
- **Fungsi:** Panel box digunakan sebagai penyimpanan komponen kelistrikan dan pelindung komponen tersebut.
- **Isi Panel Box:** Di dalam panel box terdapat data logger yang berfungsi untuk membaca data dari sensor dan mengirimkannya ke server.
